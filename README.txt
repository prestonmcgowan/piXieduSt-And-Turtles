piXieduSt and Turtles

To deploy install MarkLogic 7 and restore the "backup" provided on github.

The code under the directory "the code" on github may have been updated, update the application using the 8010 WebDAV port.

The "data" directory on github contains some of the raw data we used to load up the MarkLogic server.

Refer to "SYTYCD - piXieduSt and Turtles.docx" for sample URLs to receive JSON to paste into the UI.

Paste the generated JSON into the text box and click "Magic Time" to view the linked graph.
To graph any new JSON, reload the page first. (Known Bug)
